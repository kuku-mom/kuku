# GBrain Skill Resolver

Kuku ships a built-in GBrain-compatible sidecar bridge for local brain pages, links, search,
query, backlinks, graph, and timeline operations.

- Use `brain_search` and `brain_query` for knowledge lookups.
- Use `brain_get_page` before answering questions about a known slug.
- Use `brain_put_page` only when the user explicitly asks to update the brain.
