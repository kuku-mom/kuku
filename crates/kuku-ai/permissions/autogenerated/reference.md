## Default Permission

Allows the AI plugin commands.

#### This default permission set includes the following:

- `allow-ai-new-session`
- `allow-ai-send-message`
- `allow-ai-cancel`
- `allow-ai-get-config`
- `allow-ai-set-config`
- `allow-ai-reset-state`
- `allow-ai-list-tools`
- `allow-ai-resolve-approval`
- `allow-ai-register-proxy-tool`
- `allow-ai-unregister-proxy-tool`
- `allow-ai-submit-proxy-tool-result`
